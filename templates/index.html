Handwritten Digit Recognition

Made with Flask, Sci-Kit Learn and JavaScript, Containerized with Docker, hosted on AWS

For the best results, try to be as centered as possible and avoid small digits! Please note that this project is just a demo and is not fitted for professional use or production due to its inconsistencies. The model was trained on the TMNIST dataset, which features single digits from thousands of different typefaces in the form of 28x28 pixel images. Due to the nature of the dataset and the limitations of SKLearn, some incostistencies may occur when predicting certain numbers, especially 9. Improving on this functionality is a priority in the forseeable future. For more information about the implementation of this project, visit its GitHub page .
